# PinPic 图片拼接编辑器 - 使用说明

## 概述

PinPic 是一款图片拼接编辑网页应用，支持将多张图片进行高质量拼接，并提供丰富的边框自定义选项，最独特的在于高斯模糊背景选项。

效果示意：![PinPic-20251124_234556|100](https://github.com/user-attachments/assets/ca2ab060-e952-4b6d-8e96-c16ccf370548)

## 功能特性

### 核心功能

- **图片拼接**：将多张图片按原比例竖向拼接
    
- **边框风格**：三种边框风格可选
    
    - 纯色边框：自定义颜色
        
    - 已选图模糊：使用已选图片生成模糊背景
        
    - 自定义边框：使用自定义图片作为背景
        
    
- **高质量输出**：保持原图画质，以及自定义压缩设置

## 系统要求

- 支持HTML5的现代浏览器（Chrome、Edge、Safari等）
    
- 建议使用手机或平板设备获得最佳体验
    
- 需要启用JavaScript
    

## 注意事项

- 选择过多高分辨率图片可能导致处理时间较长
    
- 输出超高分辨率图片时请确保设备有足够内存
